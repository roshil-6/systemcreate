# Background Image Setup

## Current Setup
The login page uses a high-tech office image from Unsplash.

## Using Your Own Image

### Option 1: Local Image
1. Place your high-tech office image in: `client/public/images/office-background.jpg`
2. Update `client/src/pages/Login.css` - uncomment the local image section

### Option 2: Online Image URL
Replace the Unsplash URL in `Login.css` with your image URL.

## Recommended Image Specifications
- **Size**: 1920x1080 or larger (Full HD or 4K)
- **Format**: JPG (for smaller file size) or PNG
- **Theme**: Modern office, tech workspace, corporate environment
- **File Size**: Keep under 1MB for fast loading
- **Aspect Ratio**: 16:9 or wider

## Free Image Sources
- Unsplash: https://unsplash.com/s/photos/modern-office
- Pexels: https://www.pexels.com/search/office/
- Pixabay: https://pixabay.com/images/search/office/
